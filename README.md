Harvest Plan Scheduler

This is a simple Python project used for managing and planning crop harvest dates. 
The user can add crops with their harvest date and location. The program stores 
the information and shows a complete harvest schedule.

Features:
- Add crop name
- Add harvest date (DD-MM-YYYY)
- Add field/location name
- View full harvest plan

Python Concepts Used:
- Lists
- Dictionaries
- Functions
- Loops
- Input and Output

How to Run:
1. Open terminal or command prompt.
2. Run: python code.py
3. Use the menu to add crops or view the schedule.

This project is easy to understand and useful for learning Python basics.
